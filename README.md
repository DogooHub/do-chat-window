# do-chat-window

## Install

```bash
npm install --save do-chat-window
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'do-chat-window'
import 'do-chat-window/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
